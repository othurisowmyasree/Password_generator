# Password_generator
"Developed a Password Generator using Python and Tkinter that creates strong, random passwords based on user input. The app features a dark-themed, user-friendly GUI with options to generate and copy secure passwords instantly."

 🔐 Password Generator (Python + Tkinter)


---

 ✨ Features
- Generate strong and random passwords
- User-friendly GUI built with Tkinter
- Dark-themed interface
- Option to copy password instantly
- Customizable password length

---

🛠️ Technologies Used
- Python
- Tkinter (GUI Library)
